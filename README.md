# Frontend Studies 2026 — React 19 + Angular 21

A 60-step deliberate practice plan to reach senior-level proficiency in both React and Angular, building the same features in parallel to internalize concepts over syntax.

## Stack

- **React 19** + Vite + TypeScript
- **Angular 21** + Standalone Components + Signals

## Structure

```
.
├── react-studies/      # React 19 daily exercises
├── angular-studies/    # Angular 21 daily exercises
└── PROGRESS.md         # Daily learning log
```

## Methodology

- **Tiny problems**: each day = one focused, small win
- **Visible progress**: every commit ships something I couldn't do yesterday
- **Question-driven**: I never start a session without a clear question to answer
- **Identity-first**: I'm not "learning" these — I use them
